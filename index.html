Flexx Image Magnifier

The Flex Image Magnifier features relies on basic CSS Styling and CSS Transformations to achieve it's effect.


To use this feature you will need to do the following:
  1. Create a Div (or semantic tag....e.g., section tag) and assign class="flexx-imageSet".This will ensure appropriate CSS transformations are applied to your image.
  2. Create an unordered list that will hold your images. Each image will be a list item and you will need to assign a class of "flexx-imageMagnifier" to each each image tag.

Here is a html sample that illustrates how you would implement the aforementioned: 

                      
                 <section class="flexx-imageSet">
                    <h3>Set of images</h3>
                    <p>Rollover an image to magnify it.</p>
                    <ul>
                        <li><img src="images/01.jpg" class="image-magnifier" alt=""></li<
                        <li><img src="images/02.jpg" class="image-magnifier" alt=""></li>
                        <li><img src="images/03.jpg" class="image-magnifier" alt=""></li>
                        <li><img src="images/04.jpg" class="image-magnifier" alt=""></li>
                    </ul>
                </section>
* Note: a <section> tag was used in this example but you could use a <div> if that is your preference.
